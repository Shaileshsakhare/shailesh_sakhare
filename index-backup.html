Projects

Brillio Technologies

Policy Portal(IT Portal)

Policy portal enables client to manage the policies of different departments of the organization. Policy portal is umbrella of different portals or sub sites like GPS, RDGPS. Policy portal have different provisioned portals. Each portal has their different rules, guidelines, procedures and policies. Each subsite is provisioned from the admin site from SharePoint which creates subsite in SharePoint and the new infrastructure for sub-site in S3. Different site content can be published easily using the parallel publishing approach and synchronized using Maria DB database. Site List, Site Cache concept creates using time stamp updates the content in the site in a single refresh.

BMS 360 (Insite)

Portal gives 360 view or insights of the organization. Portal details about the published articles, news, videos and picture gallery about the recent activities of the organization. Portal showcases number of views, trending articles, stock market value of the company, news from the social media related to the organization. Portal has the facility to provision 72 websites in different languages.

IT Performance Dashboard

Dashboard application that displays the data related to the organization in the form of graphs and charts. Graphs contain the internal performance data of different departments of the organization.

IIONv2.0

II-ON is a mobile application to facilitate and better enable connectivity and collaboration across the Network centers and with BMS. It includes collaboration of SharePoint, Node JS and Cordova Application.This phase was included with further improvements related to application. The mobile application developed by Brillio was hosted on AWS (Amazon Web Services) and the data for the website was coming from SharePoint 2013.We used Node JS to query data from SharePoint and thereafter push the data to s3 (AWS container to store data). We have also used Node JS to create S3 service for the mobile application to download data securely from s3.

Research and development global policy simplified (RDGPS)

BMS R&D team intends in evolving its current compliance program and subsequent procedural documents from a rules-based to a more principles-based approach. The primary objective of this project is to evolve BMS tools and documents from their current format and content to more clear, concise (principles-based) and accessible tools available on a one-stop content delivery platform.
